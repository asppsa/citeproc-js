CiteProc::JS Engine
===================

CiteProc::JS Engine is a CSL 1.0 compatible engine for the Ruby cite processor
gem [citeproc](); it is implemented as a thin Ruby wrapper around Frank G.
Benett's awesome [citeproc-js]().
